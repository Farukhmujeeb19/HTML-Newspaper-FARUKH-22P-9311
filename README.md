## Project Description
This project recreates a historical Urdu newspaper front page using only HTML.
The layout uses semantic tags and table-based columns to visually resemble the
original newspaper structure.

## Features Implemented
- Semantic HTML5 tags (header, nav, article, section, footer)
- Three-column table layout
- Images with alt attributes
- Proper heading hierarchy
- Lists and structured layout

## Challenges Faced
Designing a newspaper-style layout using only HTML without CSS was challenging,
especially managing column alignment and spacing using tables.
